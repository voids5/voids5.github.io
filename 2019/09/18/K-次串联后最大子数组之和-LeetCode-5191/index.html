K 次串联后最大子数组之和(LeetCode-5191)

发表于 2019-09-18 分类于 C++ 阅读次数： Valine：

解题思路：lmax表示单个数组的最大前缀和，rmax表示单个数组的最大后缀和，maxx表示单个数组最大连续和,sum表示单个数组总和
1.如果n==1;返回maxx即可
2.如果n>1;如果sum>=0;比较maxx与sum*(k-2)+lmax+rmax.
如果sum<0;比较maxx与lmax+rmax.
